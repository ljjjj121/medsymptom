# MediGuide - Medicine Recommender

A web application that helps users find appropriate over-the-counter medications based on their symptoms.

## Features
- Select multiple symptoms
- Get personalized medicine recommendations
- View dosage information and warnings
- Mobile-friendly design

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Use
1. Select your primary symptom from the dropdown
2. Add additional symptoms if needed
3. Click "Get Recommendations"
4. Review the suggested medications and their details

## Disclaimer
This tool is for informational purposes only. Always consult a healthcare professional before taking any medication.
